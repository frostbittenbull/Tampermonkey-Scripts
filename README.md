# 🐒 Tampermonkey Scripts
Коллекция пользовательских скриптов для [Tampermonkey](https://www.tampermonkey.net/). Устанавливайте одним кликом и улучшайте браузер под себя.
---
## 📦 Установка
1. Установите расширение [Tampermonkey](https://www.tampermonkey.net/) для своего браузера.
2. Нажмите на кнопку **Install** у нужного скрипта — откроется страница установки.
3. Подтвердите установку.
